MiniMax Agent 积分规则

更新日期：2025年9月9日

本《MiniMax Agent积分规则》（"本规则"）将向您说明MiniMax Agent（"本产品"）的积分使用规则，请您仔细阅读。本规则构成MiniMax服务规则的一部分。您使用MiniMax产品和服务的，还应当遵守《MiniMax用户协议》《MiniMax付费服务协议》（如涉及）等有关规定。

1. 订阅计划

本产品提供免费和付费两种服务模式，不同服务模式提供不同的权益和任务处理优先级。用户可以根据自己的需求选择最合适的服务模式，具体包括：

  • 免费用户：注册后，用户将获得免费积分来体验 Agent 的任务执行功能。
  • 付费用户：提供两种不同的订阅计划（基础版和专业版）以及三个层级的积分包。用户可以选择适合自己需求的套餐，除了对应数量的积分外，还包括高峰时段优先处理和新功能抢先体验等权益。

2. 积分类型

共有三种不同类型的积分：

  • 会员积分：每月分配给订阅会员的积分，会员终止时过期。套餐积分每月自动重置为标准数量，上个月未使用的积分不会累积到下个月。
  • 奖励积分：用户可以根据需要购买额外积分。购买的奖励积分自购买之日起一年内有效。
  • 奖赏积分：用户注册时可能会奖励一定时效期的积分。

3. 积分消耗

积分代表服务完成任务所需的资源。一般来说，更复杂的任务比简单任务消耗更多积分。例如，大规模研究项目和复杂的Web应用开发通常需要更高的积分使用量。

我们根据任务类型提供积分消耗的参考估算；但是，实际使用量可能会根据任务的具体性质和当前系统条件而有所不同，请您知悉，积分将按照实际使用量进行消耗和扣除。

当前积分消耗率示例（可能会发生变化）：

  • 分析照片拍摄地点的位置信息 - 113 积分
  • 识别x公司在 2018-2023 年间发布的 AR/VR 专利，并提供其权利要求的详细清单 - 365 积分
  • 做一个个人简历网页 - 535 积分

4. 服务升级和降级

  • 升级：订阅用户可以随时升级到更高级别的套餐，当前套餐的剩余积分将被结转并合并到新的计费周期中。顶级套餐用户也可以通过购买奖励积分来购买额外积分。
  • 降级：目前不支持降级。

5. 促销优惠和新用户福利

我们可能会不定期为新用户提供促销奖励和福利。

  • 新用户福利：新注册用户可能有资格获得免费的每日积分。
  • 促销优惠：所有促销优惠均受产品网站上显示的具体条款和条件约束。

6. 积分退款政策

如果 Agent 生成失败或内容审核失败，系统将自动退还生成过程中消耗的积分。在任何其他情况下，积分都不可退款。
